注意:不要移動熾熱戰姬的視窗，避免失敗

1.執行set_pos.exe，並將滑鼠移至禮包碼輸入窗上後按一次ENTER鍵，再來移到兌換鍵上再按一次ENTER鍵，最後按一次ENTER鍵結束set_pos.exe

3.完成!

Apex Girls Gift Code Auto-Redeemer
This is a home-made automatic gift code input tool for Apex Girls (熾熱戰姬). It was created to free your hands and save you from the suffering of tendonitis (?). Please do not use this for commercial purposes.

Download Gift Code Auto-Redeemer

Instructions
Important: Do not move the Apex Girls game window during the process to avoid errors.

Run set_pos.exe.

Move your mouse cursor over the Gift Code Input Box and press ENTER once.

Move your cursor over the Redeem Button and press ENTER again.

Press ENTER one last time to finish and close set_pos.exe.

Run auto_paste.exe.

If prompted for Administrator privileges, click Yes.

Select the Apex Girls window and wait.

(Note: If you need to stop the process midway, simply flick your mouse cursor to the top-left corner of the screen.)

Done!

Note 1: The data.txt file contains the library of gift codes. You can manually add new codes to this file in the future.

Note 2: All gift codes are sourced from the Apex Girls Discord. Special thanks to the community contributors!
